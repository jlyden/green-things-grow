Static vs. Dynamic

Static

A static webpage doesn't change - it always looks the same no matter who is looking at it. The user can click links to go to other webpages, but the user cannot change the page.

Static doesn't mean boring or ugly. You can use CSS to completely change the look of a page without building the content on the fly. See CSS Zen Garden for amazing examples of how lovely a static webpage can be.

Dynamic

A dynamic webpage is built "on the fly" - when the user visits it, the page is created on the spot, possibly uniquely targeting that user based on user-specific data.
